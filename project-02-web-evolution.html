Web Evolution

Interactive · Nature · Code

Two distinct interactive web experiences exploring the beauty of nature through code. From the vibrant Natural Wilderness to the atmospheric Misty Forest, each project pushes the boundaries of what CSS, WebGL, and vanilla JavaScript can achieve in creating immersive digital landscapes.

Process

Starting with hand-drawn CSS art and evolving into procedural WebGL terrain generation, these projects represent a journey through the possibilities of browser-based art. The first edition focuses on lush, colorful forest scenes with particle effects, while the second explores moody, mist-shrouded woodland with dynamic lighting.
