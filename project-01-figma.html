Figma Prototype

UI/UX Design · Interaction

High-fidelity interactive UI prototype designed for Wuxi Wei products. This project bridges the gap between conceptual design and production-ready interfaces, featuring pixel-perfect layouts, micro-interactions, and a cohesive design system built entirely within Figma.

Process

The design journey began with extensive user flow mapping and wireframing, followed by iterative prototyping sessions. Each screen was crafted with attention to spacing, typographic hierarchy, and color harmony — ensuring the final product feels both intuitive and visually refined.
